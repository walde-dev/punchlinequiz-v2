# Distribution Strategy

## Why Distribution Matters More Than Features

v1 had 40-50 users from ONE Reddit post. The product worked. The distribution didn't. v2's entire design is optimized for distribution.

## The Core Insight

German hip hop fans are already gathered in communities. You don't need to find them — you need to show up with something fun in their space.

## Distribution Channels (in order of priority)

### 1. Reddit — r/GermanHipHop
- **What to post:** "ich hab ein [Artist]-Quiz gebaut. schafft du 10/10?"
- **Format:** Link post with a screenshot of the quiz
- **Timing:** Evening (8-10pm German time), weekdays work better
- **Why it works:** Already proven with v1. ONE post → 40-50 users.
- **Key:** Don't spam. One post per artist max. Engage in comments.
- **Subreddit:** https://www.reddit.com/r/GermanHipHop/

### 2. TikTok / Instagram Reels
- **What to post:** 30-second screen recording of quiz gameplay
- **Format:** Show a punchline → type wrong answer → type right answer → confetti → share card
- **Audio:** Use the actual song (the punchline is from) as background
- **Caption:** "8/10 im Kollegah Quiz geschafft. schaffst du mehr?"
- **Why it works:** Music + challenge + competition = algorithm fuel
- **Hashtags:** #germanrap #kollegah #haftbefehl #rapquiz #deutschrap

### 3. WhatsApp Groups
- **What to share:** The share card image + link
- **Who:** Every German has a friend group chat with rap fans
- **Why it works:** WhatsApp is the #1 messaging app in Germany. Share cards get forwarded.
- **Key:** Make the share card so good that people WANT to send it

### 4. German Rap Forums / Discord
- **Where:** Rap forums, German hip hop Discord servers
- **What:** Post the quiz link with a challenge
- **Key:** Be a community member first, not a marketer

### 5. Twitter/X
- **What:** Post share cards, punchline challenges
- **Who:** @chiefwalde audience + German rap Twitter
- **Format:** "wer kennt diese punchline? [screenshot] → link"

## The Viral Loop

```
User plays quiz
  → Gets score (e.g., 7/10)
  → Wants to flex / challenge friends
  → Shares card to WhatsApp/Instagram/Twitter
  → Friends click link
  → Friends play quiz
  → Friends share their score
  → Loop repeats
```

Every user becomes a distribution channel. The share card is the engine.

## Artist-Specific Distribution

Each artist enables targeted distribution:
- Post Kollegah quiz in Kollegah fan communities
- Post Haftbefehl quiz in Haftbefehl fan communities
- Post Apache 207 quiz in Apache 207 fan communities
- 10 artists = 10 different community posts = 10x distribution surface

## Content Flywheel

Each quiz creates content opportunities:
- Lyric breakdowns (30-second TikToks)
- "Hardest punchlines from [Artist]" threads on Reddit/Twitter
- "Can you finish this bar?" challenge videos
- Weekly "Punchline der Woche" feature

## Week 1 Playbook

1. Ship MVP with 3-5 artists, 20+ punchlines each
2. Post on r/GermanHipHop: "ich hab ein Rap-Quiz gebaut"
3. Post 1 TikTok/Reel showing gameplay
4. Share in 2-3 WhatsApp groups
5. Watch analytics. If retention is good → double down. If not → iterate.

## Metrics to Track

- **Users:** unique visitors per day
- **Retention:** % who play more than 1 round
- **Share rate:** % who click share button
- **Viral coefficient:** how many new users does each user bring
- **Artist popularity:** which artists get the most plays

## What NOT to Do

- Don't pay for ads (organic is enough at this stage)
- Don't spam communities (one post per community, max)
- Don't build features before validating distribution
- Don't add auth walls that block the viral loop
